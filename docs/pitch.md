# Pitch do Projeto

## Nome do Projeto

EconoguIA

## O Problema

Você já leu uma notícia e ficou sem entender metade do que estava escrito?

Termos como "o Copom elevou a Selic em 25 pontos-base para conter o IPCA" aparecem diariamente nos noticiários — mas para quem está começando a estudar economia, parecem um idioma diferente.

A falta de acesso a explicações simples e confiáveis dificulta tanto o aprendizado quanto a tomada de decisões financeiras do dia a dia.

## A Solução

O EconoguIA é um assistente virtual educacional de linha de comando que responde perguntas sobre conceitos econômicos de forma simples, direta e sem inventar respostas.

O usuário digita uma pergunta em linguagem natural. O sistema identifica o conceito e retorna uma explicação clara, extraída de uma base de conhecimento curada.

Quando não sabe a resposta, o EconoguIA fala que não sabe — sem inventar.

## Público-Alvo

- Estudantes do ensino médio e universitários
- Pessoas interessadas em educação financeira
- Qualquer um que queira entender as notícias econômicas do dia a dia

## Diferenciais

- **Linguagem simples** — sem jargões desnecessários
- **Transparência** — nunca inventa respostas
- **Base curada** — 30 conceitos econômicos revisados
- **Expansível** — novos conceitos podem ser adicionados diretamente no JSON
- **Foco educacional** — feito para aprender, não para parecer sofisticado

## Tecnologias Utilizadas

- **Python** — lógica principal e interface de linha de comando
- **JSON** — base de conhecimento estruturada e fácil de editar
- **Git / GitHub** — controle de versão e publicação do projeto
