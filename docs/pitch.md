# Pitch do Projeto

## Nome do Projeto

EconoguIA

## Problema

Muitas pessoas encontram dificuldades para compreender conceitos econômicos presentes em notícias, conteúdos acadêmicos e análises de mercado.

A falta de compreensão desses conceitos pode dificultar a interpretação de acontecimentos econômicos e a tomada de decisões financeiras.

## Solução

O EconoguIA foi desenvolvido para atuar como um assistente virtual educacional capaz de explicar conceitos econômicos de forma simples e objetiva.

O sistema consulta uma base de conhecimento própria e fornece respostas claras para auxiliar no aprendizado.

## Público-Alvo

- Estudantes;
- Universitários;
- Interessados em economia;
- Pessoas que buscam ampliar seus conhecimentos financeiros.

## Diferenciais

- Linguagem simples;
- Fácil utilização;
- Respostas objetivas;
- Estrutura expansível;
- Foco educacional.

## Tecnologias Utilizadas

- Python
- JSON
- Git
- GitHub

## Benefícios

- Facilita o aprendizado;
- Centraliza conceitos econômicos;
- Evita respostas inventadas;
- Pode ser expandido para incluir novos conteúdos e funcionalidades.
